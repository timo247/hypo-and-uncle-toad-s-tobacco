Mini Kboom-Js flappy bird with hand-made animations

About

This is a mini game made with the javascript library Kaboom. The source code can be found on this github repository.
You may find the mobile version very difficult, since the game has primarly been designed for desktop purpose. Pay attention to the fact that as soon as you press enter or click "rejouer", the game starts.

Commands

  • Space (desktop): jump
  • M (desktop): mute/unmute sound
  • Touch (mobile): jump
